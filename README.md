# Chad Toney - Digital Business Card 💼

A modern, responsive digital business card website showcasing professional experience, skills, certifications, and personal interests.

## 🌟 Features

- **Responsive Design**: Mobile-friendly layout that looks great on all devices
- **Microsoft-Inspired Theme**: Clean, professional color scheme inspired by Microsoft's design language
- **Comprehensive Sections**:
  - Professional summary and contact information
  - Social media links (LinkedIn, GitHub)
  - Skills & expertise (Power BI, Fabric, Tableau, SQL, Azure)
  - Career timeline with highlights
  - 6 certifications (4 Microsoft and 2 Scrum)
  - Education (MBA + BA in German Studies)
  - Personal section (family, hobbies, German fun fact)
- **Smooth Interactions**: Hover effects and animations for better user experience
- **Print-Friendly**: Optimized styles for printing

## 🚀 Getting Started

### Option 1: Open Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/chadtoney/chadtoney.github.io.git
   cd chadtoney.github.io
   ```

2. Open `index.html` in your web browser:
   - **Windows**: Double-click `index.html`
   - **Mac**: Right-click `index.html` → Open With → Browser
   - **Linux**: `xdg-open index.html`

### Option 2: Deploy with GitHub Pages

The site is automatically published at: `https://chadtoney.github.io`

### Option 3: Use a Local Server

For development with live reload:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (install http-server globally first)
npx http-server

# Using PHP
php -S localhost:8000
```

Then open `http://localhost:8000` in your browser.

## 🎨 Customization

### Updating Content

1. **Personal Information**: Edit the header section in `index.html`
2. **Skills**: Modify the skills grid section
3. **Career History**: Update the timeline section
4. **Certifications**: Add or remove certification cards
5. **Contact Info**: Update email and social media links

### Styling

All styles are in `style.css` with CSS variables for easy customization:

```css
:root {
    --primary-blue: #0078d4;      /* Main brand color */
    --dark-blue: #004578;          /* Darker accent */
    --accent-teal: #00b7c3;        /* Secondary accent */
    /* ... more variables */
}
```
